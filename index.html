Built-In Quick Sheet

Keep this open while learning. You can complete this course using only these basics.

  • console.log("text") prints text to Output.
  • const name = "Sam" stores a value that should not change.
  • let count = 0 stores a value that can change later.
  • if (condition) { ... } else { ... } chooses between two paths.
  • function greet(name) { return ... } creates reusable code.
  • const arr = [] makes a list. arr.push("x") adds to the list.
